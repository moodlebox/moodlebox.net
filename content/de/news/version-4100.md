Das Image [MoodleBox Version 4.10.0][release], veröffentlicht am 28. Juni 2025, basiert auf Moodle 5.0+.

Hier sind die wichtigsten neuen Funktionen der Version [MoodleBox 4.10.0][release]:
- [Moodle][moodle] [version 5.0+][moodle5]
- Basierend auf der neuesten Version von [Raspberry Pi OS Lite][rpios] vom 13. Mai 2025
- Eine neue Version des [MoodleBox-Plugin für Moodle][plugin]
- MoodleBox kann den Internetzugang mit einem Smartphone über USB-Tethering ermöglichen
- Weitere Verbesserungen und Fixes

Die MoodleBox-Installation erfordert die Verwendung des [Raspberry Pi Imager][rpi-imager], der für Sie automatisch das MoodleBox-Disk-Image herunterlädt.

&Gt; [Weitere Informationen zur MoodleBox Installation][install]

Alle Informationen zu dieser Version finden Sie in den [Release Notes auf Github][release].
