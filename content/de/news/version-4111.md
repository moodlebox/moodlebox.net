Das Image [MoodleBox Version 4.11.1][release] war am 28. Juni 2025 veröffentlicht. Es basiert auf der neuesten Version von [Raspberry Pi OS Lite][rpios], mit Debian Trixie, und bietet Moodle 5.1+ und seine neue Ordnerstruktur.

Hier sind die wichtigsten neuen Funktionen der Version [MoodleBox 4.11.1][release]:
- [Moodle][moodle] [version 5.1+][moodle51]
- Basierend auf der neuesten Version von [Raspberry Pi OS Lite][rpios] vom 4. Dezember 2025, mit Debian Trixie
- Weitere Verbesserungen und Fixes

Ein grosses Dankeschön an Ralf Krause für seine Hilfe beim Testen dieser Version!

Die MoodleBox-Installation erfordert die Verwendung der letzten Version des [Raspberry Pi Imager][rpi-imager], der für Sie automatisch das MoodleBox-Disk-Image herunterlädt.

Alle Informationen zu dieser Version finden Sie in den [Release Notes auf Github][release].
